1. Encontrar el Desarrollador o Compañía que ha Producido Más Aplicaciones en la Tienda
sql
Copy code
SELECT p.nombre, COUNT(*) AS cantidad_aplicaciones
FROM proveedor p
JOIN aplicacion a ON p.id = a.id_proveedor
GROUP BY p.nombre
ORDER BY cantidad_aplicaciones DESC
LIMIT 1;
Esta consulta asume que tienes una tabla proveedor y una tabla aplicacion donde id_proveedor en aplicacion es una llave foránea que referencia id en proveedor.

2. Listar las Tarjetas de Crédito Próximas a Vencer en los Próximos 3 Meses y Mostrar Cuántas Compras se han Realizado con Ellas
sql
Copy code
SELECT u.num_tdc, COUNT(c.id_compra) AS cantidad_compras
FROM usuario u
JOIN compras c ON u.id_usuario = c.id_usuario
WHERE u.fecha_venc BETWEEN CURRENT_DATE AND CURRENT_DATE + INTERVAL '3 months'
GROUP BY u.num_tdc;
Esta consulta busca usuarios cuyas tarjetas de crédito vencen en los próximos 3 meses y cuenta el número de compras realizadas con esas tarjetas.

3. Listar Usuarios que Compraron Aplicaciones de Temática ‘Cocina’ y Canciones del Género ‘Electrónica’
sql
Copy code
SELECT DISTINCT u.id_usuario, u.nombre, u.apellido
FROM usuario u
WHERE EXISTS (
    SELECT 1
    FROM compras c
    JOIN aplicacion a ON c.id_producto = a.id_aplicacion
    WHERE c.id_usuario = u.id_usuario AND a.tematica = 'cocina'
)
AND EXISTS (
    SELECT 1
    FROM compras c
    JOIN cancion can ON c.id_producto = can.id_cancion
    WHERE c.id_usuario = u.id_usuario AND can.genero = 'Electrónica'
);

Esta consulta asume que la tabla aplicacion tiene una columna tematica y la tabla cancion tiene una columna genero.

4. Listar Aplicaciones Compatibles con la Versión Más Reciente de iOS y con Puntuación Superior a 4
sql
Copy code
SELECT a.nombre
FROM aplicacion a
JOIN producto p ON a.id_aplicacion = p.id_producto
WHERE a.version_ios <= (SELECT MAX(version_ios) FROM dispositivo) AND p.puntuacion > 4;
Esta consulta asume que la tabla aplicacion tiene una columna version_ios y la tabla producto tiene una columna puntuacion.

5. Mostrar Nombres de Casas Disqueras con Más de 3 Artistas y la Canción Más Vendida de Cada una
sql
Copy code
SELECT 
    cd.nombre AS nombre_disquera,
    MAX(sub.cancion_mas_vendida) AS cancion_mas_vendida,
    MAX(sub.max_unidades_vendidas) AS max_unidades_vendidas
FROM 
    casa_disquera cd
JOIN 
    (SELECT 
         a.nombre_casa_disquera, 
         c.nomb_cancion AS cancion_mas_vendida, 
         MAX(c.un_vendidas) AS max_unidades_vendidas
     FROM 
         artista a
     JOIN 
         cancion c ON a.id_artista = c.id_artista
     GROUP BY 
         a.nombre_casa_disquera, c.nomb_cancion
    ) AS sub ON cd.nombre = sub.nombre_casa_disquera
GROUP BY 
    cd.nombre
HAVING 
    COUNT(DISTINCT sub.cancion_mas_vendida) > 1
ORDER BY 
    max_unidades_vendidas DESC;
